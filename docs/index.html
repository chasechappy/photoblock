PhotoBlock Preview

PhotoBlock is a secure, visual account and password manager that uses photos and emojis for blockchain dapps and website authentication.

Photo + Emojis = PhotoBlock

PhotoBlock is the icon above and the stuff that happens when it is selected.

Everything else on this page exists to give you an idea of how PhotoBlock works when embedded into an app or website created by other developers.

PhotoBlock can be used by anybody, but is designed primarily for non-technical users. To see PhotoBlock in action, choose a supported platform, then select the PhotoBlock icon:
